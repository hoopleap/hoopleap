《时光事务所》故事简介

##早期版本简介：

近代一个海港都市，主角是贪财的私人侦探，故事的开始是侦探在一次任务中发现自己与一个刚去世的富二代在早年换了身份，以为自己马上要巅峰了，却最终搞错落得一场空，还险些被接下来故事的导火索是侦探跟踪盗贼却发现了一个移动旅馆“时光事物招待所”。

侦探在这里认识了一只松鼠前辈（本体是百年天才小萝莉），李大爷（萝莉的马，最近当过狗和猫，同样活了上百年），飞鼠（本体是纯真操蛋熊孩子）。侦探成为了事务所的一员，侦探在这里与松鼠一同帮助他人交换身体完成任务，事务所里提供两种服务：交换灵魂体验和交易生命时间。听上去不可思议，而这都依赖事务所里一个神奇的魔蛋，这里住着很多有意思的租客（和小动物互换了身体的人），大家帮他人体验交换灵魂的任务就可以从魔蛋获得更长的生命时间，所以招待所里有很多活了上百年的“老玩家“，而平时都“住在“小动物体内，人类本体平时都装着小动物的灵魂，带着一个倒着走的手表在旅馆里熟睡（可以有一些历史名人，比如希特勒必入特斯拉比如李白比如……）。人们想活的更久，就在这里给自己赚取时间（应该还是比较好理解的，千与千寻，还有很多影片甚至宝青坊可能都有些类似设定）。

时光事务所——为什么写这个故事

关于立意，最早这个故事的初衷来自于 ”时间” 这个话题，这些年大家所接受到的信息量是20年前没有网络和智能手机的时候的几十倍，这些年的世界的变化速度也是前所未有的，而生命”时间“是有限的，现在给大家提供了越来越多去体验他人的人生的机会，比如”看电影“，再进一步放大欲望，电子游戏，再继续发展，VR这样的沉浸式体验……大家都在关注怎样在有限的生命里去获得尽可能多的人生体验。现在的西方故事多在关注个人与社会 的关系冲突，比如《黑客帝国》、《西部世界》；东方的故事的审美倾向明显更关注个人和个人甚至自己的关系冲突，中国流行文化里”武侠”就是典型，是成功输出的文化，现在在文化输出的是玄幻，最火的“爽文”价值观还在有仇必报（所以”爽“啊）的稚嫩阶段，影响范围很大，但层次不高，现在火的快的，去的也快，达不到经典的影响力，就像45度仰望星空那种，做的好是商业主流，但也怕变成文化主流里的不入流……看的人长大了也就索然无味了，反而是《哈利波特》这种奇幻类型的持续性比较强。

2016年490亿票房的类型分布数据：

爱情影片依然数量最多，动画片占据了总票房的7分之1，感觉未来会是个稳定数字。

我们说的二次元文化其实是一种典型的东方人文关怀，一种青春惆怅，在纠结自身问题），讨论青春期自己的内心冲突和玩笑性质的外界冲突。这种意识形态之所以在发展也是很有中国当下的必然性的，最浅显的来讲就是信息变得比以前传播的快得多，更加透明，而这一会让人更加需要自己的内心世界堡垒，我们的年轻人关心的东西其实越来越和息息相关相关了，而《时光事务所》就是这样的”最小化奇幻“类型（其实是魔幻了，国内大家叫的比较乱），内核讲的是个人和个人还有自己的内心冲突，外在是动作、爱情、推理和犯罪等元素。在”时间“命题下本质还是讲”体验 “，想要更多的时间是因为巨大的信息让人的欲望越来越膨胀，进而大家想得到的东西就更多。而体验的终极是”让人获得当下自身条件所无法获得的人生“，这和中国人的集体意识有关，比如成仙了（典型的修仙故事），穿越剧等，本质是”跳出自我的框架，超越当下环境中自我的束缚“，在这个动因驱动下，大家接受了各种同类泛滥的题材，直到看腻了……我的思路是找到东方故事里成功吸引到大家最核心的因素（过去做到流行并输出了的文化是金庸代表的武侠，我不大看好玄幻现在的发展模式，虽然本身限制小起点低，大量游戏行业和有资本的人看到了机会，随便套一个没有内涵的故事，”爽”完后，日渐成长的观众已经开始排斥），年青一代所接受的是能”嗨起来“的本质。70后，80后，90后，消费意识形态都在不同的阶段。近10年的巨大变迁里，有人了解更多的过去，有人看到变化的过程，有人得到变化的结果。现在的90后”不纠结不愤怒”，起码我看到周围的情况，愤怒的是一部分70后，纠结的是一部分80后。当然未来会更多元化，对我们而言，找到商业价值和文化价值的交集是最重要的工作。

数据是日本票房，柯南的受众是偏向女性的，海贼是偏向男性热血的，反应在票房上海贼不是对手，这是值得研究的一个现象。

我锁定目前年轻观众关心两大命题：”自我实现“和”情感关系“，国内90后在经济上不是支柱，但消费上是主流，钱怎么赚不敢说，但怎么花是90后和00后说了算，大家接受信息很快也很多，欲望处于膨胀得不到满足的状态，而在父辈们和以往的意识形态的压力下，大家的话题就落在了爱情，工作，自我满足，社会回报等等这些事情上面，《疯狂动物城》是”自我实现“为主”情感关系“为辅，《音符》两个世界主讲”自我实现“和”情感关系“两条线穿插，《白蛇》最早版本是”自我实现“的白蛇视角，现在改方向是纯粹的”情感关系“的阿宣视角。相对而言，大家最关心的不是单一的自我实现和单一的情感关系，而是把两者放到一起的矛盾和冲突。《音符》这个故事我的遗憾就在于两条线缺乏这两者之间的冲突，其实《疯狂动物城》里兔子实现自己和狐狸的感情之间有这种冲突体现，这个是一个很好的基础，在接下来开发的故事里我会强调这个方面。

某视频网站对当下观众做的一个调查问卷，提炼出来是要满足两点：”首先要有娱乐性的故事和特色鲜明的角色，然后在看完之后要有一定深度的思考。“这意思先是需要用套路吸引大家进来，然后给大家建立足够的期待看下去，最后要给大家一些值得思考的空间和讨论传播的价值，前一点是可学习可复制的套路吆喝，后一点是决定了口碑和价值的关键因素。在这样的思考前提和当下，我基本确定了这样一个范围，在未来十年我会关注并探索的命题：关于时间、体验、梦、虚拟现实、生存和商品社会。

  1. 一种是奇幻的身份互换故事：《三国大穿越》和 《时光事务所》
  2. 一种是近未来科幻的人工智能和多重人格：《少女X黑客》（多重人格少女和多职业身份大叔故事）
  3. 一个比较大的未来话题：《花花世界》（故事大概念：人类的未来需要更大的生存空间，地球表面已经不能再过度开发了，人类分为两大派系，一类人主张限制人口往地下移民，另一类向外太空探索生存空间……有个故事里是地下世界的少年和探索太空的少女的一段穿越光年的爱情史诗……在这个概念下可以有很多三百六十度环视宇宙欢喜奔袭而来的脑洞篇章）
  4. 梦的主题：”记梦器”的故事……一直以来，人类有一种将梦记录保存的手段，这是一种从轩辕时代就存在的技术，下一次进入梦境可以接着读档继续进行，并将梦与梦连续起来，那么我们，在现实世界和梦的世界分成两个对立的世界，同时存在，再进行修改……也许可以把这个主题放到《少女X黑客》这个科幻故事里。

《时光事务所》是我现在正在开发的故事，主角是一名侦探，大家熟知的侦探故事一般都有比较混乱的时代背景，对侦探的印象多停留在拿着放大镜，叼着烟斗的样子。而严格来讲《时光事务所》不是这样一个侦探故事，但我希望有一种侦探片基调和穿插进来的推理情节，“侦探”身份具备一种天然的旁观视角，观众对这样的类型有一个约定俗成的印象：影片中角色获取的线索和观众获取的线索相同，有很多段落里这样做可以提高观众对故事情节的注意力，因为大家有一种寻找到线索看清楚真相的”解题的快感“。更多的信息量产生刺激的体验，这种刺激和动作刺激不同，男性和女性在这个层面上是对等的，推理类型的女性观众甚至超越男性（看上去好厉害啊！好帅啊……），他人视角的”错位感“也是一种典型的“梦的视角”，所以主角是侦探我是有意为之。 然后就是对受众人群的考虑，我想做的这种主题和 主题本身是”交换“身份这种魔幻类型，而把人变成小动物的拟人化表演是典型的奇幻，故事里的主线冲突是主角要收集到很多“成功案例”，直到达成约定解除魔咒，变回人来。所以故事定位在当代都市，就在大家身边的感觉，紧跟潮流，因为本身故事和身份有关，我希望尽可能让大家能带入到自己身边。

讲什么和怎么讲

我这里说的东西可能不属于书本上的直接释义，虽然我们会去研究很多书本和资料，但我最相信的还是通过自己看电影，自己做电影，还有身边的人的反应所总结而来的东西。 前面讲了很多市场和取材方面的东西，也就是“讲什么故事”，接下来我讲的是取材之外，“故事怎么讲”。 一个创意集体（制片公司）有人专门研究市场数据，这是方向性战略性的东西，决策层会选取题材方向，接下来故事怎么讲也是有专门的格式的。 让故事产生认同感（代入感）和对商业电影的成功有直接影响。这句话是没毛病的，哪怕是我们没有经历战争，但看战争片同样可以让人产生认同感（代入感），这是通过电影里生动的角色刻画和环境建立了故事和观众的桥梁。基本上符合这个公式：  认同感=移情 X 可信

茶宠：现在和未来 —— 前篇1

最近眼疾，文章写了很多次，啰嗦了一些，就当我们展开想象的前提，前篇1是思考和回顾，前篇2

大家看戏的本质还是”体验”，看完后生活并不会有什么改变。电影的本质是用视听提供一种”无风险”体验他样人生的娱乐方式。现今文化娱乐价值观里，并不纠结”对与错”，但对”美”是有广泛诉求的。”道德”在个体层面其实是反人性的，为了保护人在群体的利益而必要存在的，而主流人群去看电影是要看”个人”东西，是要进入另一个世界，”美”就是一种相对个人的东西，也是一种对生活的高度概括，动画在商业考量上相对实拍电影在国内的劣势有很多，但这种”虚构的距离感”和一些美的本质更接近，这也是动画的魅力所在，是动画应该着重去传达的信息。

所以，无论说星海的审美太日式，太矫情，最终还是要承认，很多观众看电影的本质恰好被切中了，这就好像大家对糖水片的需求是微单销量的保障，同样能拍糖水为什么还要去买单反呢？当然这里说的”美”不仅美术，还包含了普世的人类情感，只是视觉先于情感。故事为王不是说故事比技术重要，而是先后关系，敢说故事为王的可能只有皮克斯了，技术领先的无冕之王喜欢说艺术…… 界定”文艺片”和”商业片”，从结果看是能赚钱和不能赚钱，从创作来看是审美的范围，高不可沾就是文艺片，这是个梭子型的模型，最多人可欣赏的在中间，我们要做的是高于观众一点点，等于或者低于普通观众的审美会被叫烂片，商业上能成功的”爆款”不是”美的终极”，更不是丑得稀奇，之前提到过迪士尼的审美，其实就是一种经典的标准，”恰当的美好”，”恰好高出观众一点”，就像看悬疑，让观众能跟上，再给一点惊喜和意料之外。

看个数据：

  • 全球票房前几名：阿凡达、泰坦尼克、复联系列、速度与激情系列……

  • 华语票房前几名：美人鱼，捉妖计，寻龙决，西游系列，囧系列……

    研究这些大众审美和一些数据，从中可以看到很多有意思的东西，比如全球来看大家都喜欢”强世界观”的影片（不讨论类型），除了泰坦尼克和囧系列，严格来讲泰坦尼克也是强世界观。国内来看，排除了进口片，所以囊括本土文化的国内市场对强世界观的接纳范围可能超过全球市场。而”强世界观”在”类型”的基础上再细分：比较清晰的是西方审美早期偏向”爱情”题材，现在喜欢看热闹——”大家一起搞事情”（复联、速七）；国内观众一直是”看耍宝逗乐和热闹”、”看猎奇和刺激”、”看偶像”，东西方的共性集中在幻想题材（幻想题材一定是强世界观），然后就是”系列片”，加上进口片那国内也是喜欢看——”大家一起搞事情”（其实就是看热闹），而这种类型最好是做系列片。，茶宠如果第一部比较成功的话可以试一试。当然我们看历史是为了探究本质，回顾过去的目的是要把握未来，我相信创作的根源是来源于对这些本质的理解。

回顾：

记得星海的审美在10年前的几个短片可能只是我读书时候动画学院的少数人会关注，普通观众看不到，他的文艺小资情调弄不好就很矫情，水平也只属于”清新的学生作品”，然后的《秒速》在二次元和动漫者里有小部分人追捧，依然属于小众。《言叶之庭》延续《秒速五厘米》的特点，讲的成人的故事，技巧更加娴熟，风格唯美，关注生活场景细节的PS技巧和糖水画风算是做到了极致，唯美动态mate painting毫不掩饰炫技，故事比较伤感……《追逐繁星的孩子》，严重模仿宫崎骏，场面更是宏达，再到现在的《你的名字》（能获得大众追捧也都在整体审美之上，主要是完整度和特点鲜明），同样MV拍法的还有国内的《小时代》，但小时代有点偏离正常价值观了，不作为动画片参考范围。

《你的名字》宣发历程：

  1. 国内视频网竞价时：优土三倍于B站出价购买了《你的名字》，”不小心”泄露了低清版本（不是枪版），日本制片方很失望，推迟了几个月到年底光线发行。星海当时的卖点在小圈子里是唯美清新的”画面”，其实大家去网上看枪版的心理只能是”看故事”，毕竟渣视听，而看”画面”和”看故事”的心态是矛盾的，所以下载了的人也并不多，集中在粉丝群，而好片真的巴不得漏出来让大家酝酿一番。

导演阐述的两个阶段

看阐述对象，早期的导演阐述在针对给制片人的一个要拍的东西的核心想法报告，这个报告除了导演自己的能力之外还很考验制片人的判断能力，有些导演真的阐述的不怎么样，但一个牛逼的制片人可以识别这个项目背后的价值和魅力（这是从制片人角度去看这个问题，不是导演角度），而我们发现，好莱坞最优秀的制片人（赚钱的和拿奖的）都有导演经验，但不会在一个影片里又是导演又是制片人，会有意分成两个人在一部影片中，因为这样才能形成良好的制约平衡的机制。

第二阶段的导演阐述，是影响影片实际拍摄的部分 。这其中有6个要素需要阐述和传达给整个制作团队和剧组，期中每个部门的需求不同，为了保证执行效率，导演会有针对性给每个部门传达这几个要素里关键的部分。

这里具体的“导演阐述”有两个作用，一个是检测和梳理自己的思路，二是如何转化成可执行的导演语言。

完整要点有6个：

  1. 视角
  2. 情绪状态
  3. 关注点
  4. 画面要素
  5. 时间节奏
  6. 声音要素

不考虑其他可控因素，比如成本，Previz可以预览检验很多要素，对导演和制片来说是很好的检验工具，在技术进步的将来，可以在实现手段上越来越多替代原本故事板的预演工作。

其中前3个是可以提炼出来给故事板的需求：

视角、情绪、关注点

视角的意义 ，视角就是我们把观众放到什么角度去看，带入哪个角色？主观还是客观？其实就是小说中的第几人称的问题，是第一人称还是第三人称？

  • 以哪个角色的视角展开
    • 为什么是这个视角？意义？
      • 视角是镜头之上的宏观意义
      • 视角就是摄影机所处的立场
      • 是艺术表现的范畴，导演或故事艺术家的选择，也正是这种选择造就了叙事风格的差异。
    • 几种视角？
      • 视角的分类
        • 第一人称
        • 第三人称
        • 全知上帝视角
        • 只用第一人称和第三称是因为第二人称很奇怪
      • 主观视角，第一人称
        • 单一角色视角
        • 多重视角转换
      • 客观视角，第三人称
      • 全知上帝视角
      • 奇观和视角转换
